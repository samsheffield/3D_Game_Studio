Warmup Prototype (Due: Feb. 03)

This is a 2 week prototype exploring some mechanic or concept that you might like pursue further.

  • This is a prototype. Energy should be put towards experimentation, asking questions, and taking risks; not refining or polishing.
  • It can be rough, but it should feel complete. Make sure all the pieces are in place for it to make sense.
  • This can be completed individually, or in pairs.
  • Must use Unity's 3D workflow.
  • This will be used to facilitate conversation & group formation leading into the core project.
  • Participating in the Global Game Jam? See the note below about using that as partial credit.
